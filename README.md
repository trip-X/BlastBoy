# Blaster Boy

一款基于 Cocos Creator 3.8 开发的 2D 生存类动作游戏。玩家操控角色在地图中移动，击败不断涌来的敌人，收集经验升级，获取新武器和属性强化，挑战更长的生存时间。

## 游戏特色

- **摇杆操控** — 虚拟摇杆控制角色移动，自动锁定范围内敌人攻击
- **6 种武器** — 飞剑、爆裂枪、毒圈、洋葱剑、散弹枪、雷拳，各具特色
- **5 种敌人** — 蓝角兽、绿毛怪、木乃伊、黑曜切割者、翠绿独眼，随时间逐步解锁
- **升级系统** — 击杀敌人掉落经验，升级后可选择新武器、武器强化或属性提升
- **动态难度** — 敌人属性随时间增长，刷新频率和数量逐步提升

## 武器系统

| 武器 | 类型 | 特点 |
|------|------|------|
| 飞剑 | 近战/吸附 | 多段连击，高频输出 |
| 爆裂枪 | 远程/吸附 | 均衡射击，稳定伤害 |
| 毒圈 | 范围/放置 | 持续范围伤害，无需瞄准 |
| 洋葱剑 | 近战/吸附 | 重击型，单次高伤 |
| 散弹枪 | 远程/吸附 | 扇形散射，近距离爆发 |
| 雷拳 | 远程/吸附 | 高速弹道，强力单体 |

每种武器可强化 3 级，提升伤害倍率、攻击速度、子弹数量等属性。

## 敌人系统

| 敌人 | 解锁时间 | 特点 |
|------|----------|------|
| 蓝角兽 | 0:00 | 基础敌人，属性均衡 |
| 绿毛怪 | 1:00 | 低血量高攻击 |
| 木乃伊 | 2:00 | 远程射击型敌人 |
| 黑曜切割者 | 3:00 | 高攻高防精英 |
| 翠绿独眼 | 4:00 | 最强敌人，全面属性碾压 |

敌人每 30 秒提升一级（最高 30 级），属性按成长系数递增。

## 升级系统

升级时从 3 个选项中选择：

- **获取新武器** — 解锁一种尚未拥有的武器
- **武器强化** — 提升已拥有武器的伤害、攻速等（最高 3 级）
- **属性提升** — 增加攻击 / 最大生命值 / 攻击半径 / 防御（最高 3 级）

## 技术栈

- **引擎**：Cocos Creator 3.8.7
- **语言**：TypeScript
- **架构**：单例管理器 + 状态机 + 实体组件 + 对象池 + 事件总线

## 项目结构

```
assets/
├── Maps/              # 地图配置
├── Scenes/            # 场景文件（开始场景、战斗场景）
├── Scripts/
│   ├── Base/          # 基础类（单例、实体管理器、状态机、属性系统）
│   ├── Drop/          # 掉落物系统（经验、生命、磁铁吸附）
│   ├── Enemy/         # 敌人系统（5种敌人 + 工厂 + 对象池 + 生成配置）
│   ├── Enums/         # 全局枚举定义
│   ├── Player/        # 玩家系统（移动、受伤、无敌帧）
│   ├── Runtime/       # 运行时管理器（游戏、数据、事件、资源）
│   ├── Scene/         # 场景管理（战斗管理器、波次生成器）
│   ├── Stats/         # 属性系统（玩家/敌人属性）
│   ├── Tile/          # 地图瓦片系统
│   ├── UI/            # 界面（摇杆、血条、经验条、暂停、结算、升级选择）
│   ├── Upgrade/       # 升级系统（配置、权重策略）
│   ├── Utils/         # 工具函数
│   └── Weapon/        # 武器系统（6种武器 + 子弹 + 配置）
└── resources/         # 资源文件（贴图、预制体、动画、字体）
```

## 运行项目

1. 安装 [Cocos Dashboard](https://www.cocos.com/creator-download)
2. 在 Dashboard 中添加 Cocos Creator 3.8.7 编辑器版本
3. 导入本项目
4. 打开 `BattleScene` 场景，点击预览运行
